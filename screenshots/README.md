# Screenshots

Add your application screenshots here:

- `landing.png` - Landing page screenshot
- `browse.png` - Browse movies page screenshot
- `mylist.png` - My List page screenshot

You can take screenshots using:
- Windows: Win + Shift + S
- Mac: Cmd + Shift + 4
- Browser DevTools: F12 → Device Toolbar → Screenshot

Recommended size: 1920x1080 or 1280x720
