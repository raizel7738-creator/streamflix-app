## 💼 Resume/Portfolio Description

**StreamFlix - Full-Stack MERN Netflix Clone**

Developed a production-ready Netflix clone using the MERN stack (MongoDB, Express.js, React, Node.js) featuring:

• Implemented JWT-based authentication system with secure password hashing and token management
• Built RESTful API with 15+ endpoints, including protected routes and role-based access control
• Designed and implemented MongoDB database schema with user management and watchlist functionality
• Created responsive React frontend using Context API for state management and custom hooks
• Integrated TMDB API for real-time movie data with proxy layer for API key security
• Applied security best practices including CORS, rate limiting, input validation, and Helmet middleware
• Developed service layer architecture for clean separation of concerns and maintainable code

**Tech Stack**: React 19, Node.js, Express.js, MongoDB Atlas, JWT, bcrypt, Axios, Context API, TMDB API
